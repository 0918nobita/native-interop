# Native Interoperability

Call a function (from a shared library, in C) from C#, F#, OCaml or Racket

## Supported platform

- Linux

## Requirements

- .NET Core SDK 3.1 or later
- OCaml 4.08 or later
- Racket 7.2 or later

## Build

```bash
make
```

## Run

### C#

```bash
make run-csharp
```

### F#

```bash
make run-fsharp
```

### OCaml

```bash
make run-ocaml
```

### Racket


```bash
make run-racket
```

## Clean

```bash
make clean
```
